# V2 Summary

This week I did a lot of work on Lore to make it easier to scan, plan with, and read.

- Added clear `Travel` and `Discover` modes with a simple switch.
- Simplified the sidebar into a lighter control area and moved saved spots into a drawer.
- Replaced the old detailed category system with broad buckets like People, Places, Events, and Landmarks.
- Shrunk the map markers so they read as small dots instead of covering the map.
- Added a distinct current-location marker so the user position is easy to spot.
- Improved the place content pipeline so Wikipedia text is preferred when available, with Wikidata used as backup and enrichment.
- Expanded the detail panel so places open in a short state first, then show more when the user clicks `Show more`.
- Used Codex across sessions to keep pushing the same v2 work forward, including sidebar cleanup, category simplification, and Wikipedia/Wikidata matching.
